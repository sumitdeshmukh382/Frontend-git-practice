This is Day 2 Git practice

git --version - Check the version of git

git config --global user.name 'Name' - add the user name

git config --global user.email 'ouremail' - Add the email

git init - initialize the repository/project

git add . - add all the changes to the repository

git add index.html - add the changes that are made in that file to the repository

git add file-name - add the changes that are made in that file to the repository

git commit -m 'Added a button/paragraph/Page'

git remote add origin folder/file.git - Add remote repository as a origin

git branch -M master/main - create main/master branch

git branch homepage - create a branch with name home page

git merge aboutpage - Merge the aboutpage branch with current branch

git chechout aboutpage - Switch to a specific branch

git branch - show all the branches in our origin/repository

git push -u origin master - pushes the code in master branch

git push -u origin branch_name - push the code to the repository

git pull - pull/update the code in our device after merging

git fetch - pull/update the code without merging

git clone url - clone the project

git log - show the commit history

git log branch-name -commit history of the perticular branch

git log --all - commit history of all the branch

git log --oneline - show commit history in the condensed manner

git remote - shows origin is added or not

git remote -v - show the remote repository

git remote remove origin - remove the remote repoitory
